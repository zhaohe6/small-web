js基础知识补充

（1）iframe标签用于加载网页在特定的位置

（2）input 标签的value属性是用来设置文本框的内容

（3）input标签的name属性：

只有设置了 name 属性的表单元素才能在提交表单时传递它们的值，因为服务端获取表单提交的数据是通过表单元素的 name 属性的值而得到的，没有 name 属性就无法得到表单元素提交给服务端的值

（4）input的onblur属性：

​ 失去焦点事件，与onfocus相反

Ajax的常用的参数

url：待载入页面的URL地址

data：待发送的key/value参数

success：载入成功时的回调函数，有两个参数，第一个参数data返回服务器的数据，第二个参数status返回服务器的状态

Ajax请求将文本框输入的值发送给服务器

预备知识——从form表单说起

form表单收集数据，并且把数据提交到后端，其中的action属性可以填写要提交到的网址，其中的method属性可以填写get或者post来确定请求的类型；注意input一定要有name属性。

get和post请求的区别主要有两点

1.get请求的会将提交的数据显示在url的后面，post请求不会，所以post请求的更安全一些

2.get请求有数据的上限，post请求没有。
