C++中的指针和C语言中的类似，和java中类似的指针操作有些区别

格式：类名* 指针名 = new 类名 (参数1，参数2)；

C++的指针数组和指向数组的指针傻傻分不清？？

解释：指针数组，注意中心词：“数组”，这就说明，这是一个数组，但是为啥又有 “指针” 二字呢？ 这是因为这个数组里面放的是指针，就是这样！指向数组的指针，中心词是指针，就说明这是一个指针，只不过这个指针是指向数组的。

指针数组的定义：int*a[4];

数组指针的定义：int(*a)[4];
