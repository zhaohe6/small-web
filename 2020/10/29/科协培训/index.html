知识点三：循环

while do…while循环

while语句可以简单的记为：只要当循环条件表达式为真，就执行循环体语句。while循环的特点是先判断条件表达式是否为真，在执行循环体语句。do…while循环语句的特点是：先无条件执行do的循环体，然后判断条件是否成立。

for循环

for循环括号中三个表达式的作用：第一个表达式是为循环变量赋初值，只执行一次；第二个表达式是循环条件表达式，用来判定是否继续执行这个循环的循环体；第三个表达式是循环的调整，让循环变量增值，是在执行完循环体之后执行的。

二维数组

用矩阵的形式来表示二维数组是逻辑上的概念，能形象表示行和列的欢喜。而在内存中，各元素是连续存放的，不是二维的，而是连续存放的，这点务必明确。

二维数组的初始化：

1.分行给二维数组赋值

2.将所有的元素写在一个大括号内，不用嵌套大括号

3.对一部分赋值，其余的元素自动赋值为0.

4.如果对所有的元素在一个大括号内赋值，那个第一位可以不写。

const char * 和char * const

char * const 定义一个指向字符的指针常数，即const指针，实验得知，不能修改ptr指针，但是可以修改该指针指向的内容。

const char * 定义一个指向字符常量的指针，这里，ptr是一个指向 char* 类型的常量，所以不能用ptr来修改所指向的内容，换句话说，*ptr的值为const，不能修改。但是ptr的声明并不意味着它指向的值实际上就是一个常量，而只是意味着对ptr而言，这个值是常量。实验如下：ptr指向str，而str不是const，可以直接通过str变量来修改str的值，但是确不能通过ptr指针来修改。
