Get vs Post

预备知识——从form表单说起

form表单收集数据，并且把数据提交到后端，其中的action属性可以填写要提交到的网址，其中的method属性可以填写get或者post来确定请求的类型；注意input一定要有name属性。

@get和post请求的区别主要有两点

1.get请求的会将提交的数据显示在url的后面，post请求不会，所以post请求的更安全一些

2.get请求有数据的上限，post请求没有。

Ajax发送数据的五个步骤

  1. 创建对象：
var httprequest = new XMLHttprequest();

设置请求方式和请求地址：
httprequest.open("GET","liujing",true);

发送请求：
httprequest.send()

监听状态变化：
httprequest.onreadystatechange = function(ev2){
if(httprequest.readyState === 4){
//等于4就说明4个状态都完成了
    if(httprequest.status>=200&&httprequest.status<300||httprequest.status===304){
    console.log("接受到服务器返回的数据");
    }else{
    console.log("没有接受到服务器返回的数据");
    }
}
}
